# public_preview
This is small repo for public visibility of some projects.

AutoTab was a an app made years ago to fulfill the need to have a constantly rotating screen selection for multiple live-feeds at the same time. It was a quick build using command prompt to just accomplish the need without any decoration.

AutoTab Taskbar is an updated version of AutoTab that utilizes the actual Windows interface to make a smooth and professional version of the app with features more in line with a legitimate public-facing app. It makes a button on the taskbar that allows for proper pausing/continuing of the process, gives information about the time left between transitions, and is generally better looking and functioning. It could easily be expanded upon but suits my needs as far as I've had need of it to. 

Valentines is a fun and very simple Valentine's Anniversary html page made for my wife a while ago. It isn't in-depth but has lots of room for potential growth and expension. I plan to continue working on it over the years around Valentine's and keep growing it into something more complex and enjoyeable.
